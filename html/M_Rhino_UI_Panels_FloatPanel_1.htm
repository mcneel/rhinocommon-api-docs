PanelsFloatPanel Method (Type, PanelsFloatPanelMode)

Mac support: Display the specified panel in a floating window on Mac, the floating window will only contain the specified panel. Windows support: On Windows this will show or hide the floating continer containing the specified panel. If the tab is docked with other tabs it will be floated in a new container.
