GetValue(T) Method

RuntimeDocumentDataTableGetValueT Method

Checks the dictionary for the specified key, if found and the value is not null then then the value is returned. If the key is not found or its value is null then newT(Document) is called to create a new value instnce which is put in the dictionary and returned.
