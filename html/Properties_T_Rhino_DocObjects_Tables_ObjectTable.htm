The ObjectTable type exposes the following members.

Properties
  NameDescription
BoundingBox
Gets the boundingbox for all objects (normal, locked and hidden) in this document that exist in "model" space. This bounding box does not include objects that exist in layout space.
BoundingBoxVisible
Gets the boundingbox for all visible objects (normal and locked) in this document that exist in "model" space. This bounding box does not include hidden objects or any objects that exist in layout space.
